Instrucciones de Importación - Productos
Formato del archivo CSV:

El archivo debe contener las siguientes columnas (en el mismo orden):

Columna Requerido Descripción Ejemplo
codigo No Código interno del producto PROD-001
codigo_barras No Código de barras del producto 7501000000001
nombre Sí Nombre del producto Arroz Integral 1kg
descripcion No Descripción del producto Arroz de grano largo
precio_compra Sí Precio de compra 15.50
precio_venta Sí Precio de venta 20.00
stock Sí Cantidad en inventario 100
stock_minimo Sí Stock mínimo para alertas 10
categoria No Categoría del producto Abarrotes
proveedor_id Sí ID del proveedor 1
activo No Estado del producto (True/False) True
Campos Fiscales (CFDI 4.0)
clave_producto_sat No Clave del producto según el SAT 01010101
unidad_medida_sat No Unidad de medida (SAT) H87
clave_unidad_sat No Clave de unidad (SAT) E48
objeto_impuesto_sat No Objeto de impuesto (01/02) 02
Notas importantes:
  • La primera línea debe contener los nombres de las columnas
  • Los productos duplicados (mismo código, código de barras o nombre) serán ignorados
  • El proveedor_id debe existir en la base de datos
  • El archivo debe estar codificado en UTF-8 para caracteres especiales
  • Los valores booleanos (activo) deben ser True/False
  • Los campos fiscales son necesarios para la facturación electrónica
